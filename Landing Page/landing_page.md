# Landing Page Guide

## Purpose
This guide helps you use Builder Kit to quickly build high-converting landing pages for your AI SaaS product.

## Steps
1. **Choose a Template:** Start with a prebuilt layout or create your own.
2. **Add Components:** Drag and drop hero sections, CTAs, testimonials, pricing tables, and more.
3. **Customize Content:** Edit text, images, and branding to match your product.
4. **Integrate CTAs:** Use the CTA module for signup, demo, or contact actions.
5. **Preview & Publish:** Test responsiveness and deploy with one click.

## Example Structure
- Hero Section
- Features Overview
- CTA Banner
- Testimonials
- Pricing Table
- Footer

## Sample Code
```jsx
<LandingPage>
  <Hero title="AI SaaS for Everyone" />
  <Features items={featuresList} />
  <CTAButton text="Get Started" />
  <Testimonials data={testimonials} />
  <Pricing plans={pricingOptions} />
</LandingPage>
```

## Tips
- Keep messaging clear and focused.
- Use social proof and clear pricing.
- Optimize for mobile and fast load times.
