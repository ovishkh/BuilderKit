# Error Tracking (Sentry)

Catch bugs before your users do with real-time error monitoring.

## Why Sentry?
- Deep integration with Next.js (Client & Server side).
- Stack traces and breadcrumbs for every error.
- Performance monitoring.

## Installation
```bash
npx @sentry/wizard@latest -i nextjs
```

## Best Practices
- Connect your GitHub repo to link commits with errors.
- Use `Sentry.captureException()` for custom error handling.
