# Transactional Email (Resend)

Send beautiful, responsive emails using React components.

## Why Resend?
- Modern API designed for developers.
- Built-in support for `@react-email`.
- High deliverability.

## Installation
```bash
npm install resend @react-email/components
```

## Common SaaS Emails
1. **Welcome Email**: Sent immediately after signup.
2. **Password Reset**: Secure link for account recovery.
3. **Subscription Success**: Confirmation of purchase.
4. **Usage Limits**: Notifying users when they reach plan quotas.
